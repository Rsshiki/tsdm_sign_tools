# 天使动漫签到工具

基于python编写的天使动漫论坛的自动签到打工工具 目前仅实现windows专用

默认使用firefox浏览器、selenium驱动

模拟网页端操作

支持多个账号

支持自动运行

最小化可以挂在系统托盘

通过注册表实现开机自启动

PS.捣鼓捣鼓request,重新写了一版基于request的无浏览器版本,操作界面直接把上一版拿来改了一下，基本功能一致，签到及打工速度能比浏览器快许多（毕竟省了网页加载:P）。
